# Dataset

This project uses the Store Sales Time Series dataset from Kaggle:

https://www.kaggle.com/competitions/store-sales-time-series-forecasting

Due to size limitations, the dataset is not included in this repository.

## Files Used
- train.csv
- test.csv
- stores.csv
- transactions.csv
- oil.csv
- holidays_events.csv

## Instructions
Download the dataset manually from Kaggle and place the files in the `data/raw/` folder.
